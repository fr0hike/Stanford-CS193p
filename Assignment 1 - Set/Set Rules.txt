Rules:
1.) Sets are defined as:
	> Dont need same color
	> Fill must be different
		>EX: All are green, one is empty, another has squiggles, another has lines in them
	> In general, you want one thing to be the same about each card.
		"All are the same shape"
		"All are the same color"
		"All have the same number of objects on the card"
	... that is what is defined as a set
